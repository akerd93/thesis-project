Integration Framework Chatbot

Do you have questions about products, workflows or SQL functions?

Ask the Bot

This chatbot has been created as a project accompanying the master thesis Integration Framework between Natural Language Processing and UA Content based on Chatbot Technologies.

The original version of the chatbot is only available SAP internally - this version runs on mock data and does not use any SAP APIs. Therefore, the result of searches, chapters and content
will always be the same - no matter the search objective. This is intended as this is supposed to be a proof of concept.

 

Product scenario

  • Type: I'm looking for a product
  • Type your search parameter: SAP HANA
  • Choose a result

Workflow or SQL function scenarios

  • Type what you want to know: How do I delegate workflows automatically? or I'm looking for a function to convert a string to lowercase
  • Choose a result
  • Choose a keyblock/chapter

Weather scenario

  • Ask: How is the weather in Mannheim tomorrow?

Technology

  • The right skill from a chatbot framework (here SAP CAI) is mapped
  • If necessary, a full text search across Help Portal content is done
  • For the product, workflow and SQL function scenarios, users can choose from different results
  • In the workflow and SQL functions scenarios, the HTML topic is loaded from the Help Portal, they keyblocks are parsed and displayed as result
  • Choosing a keyblock shows the Help Portal content of the keyblock
  • The weather scenario runs on the MetaWeather API
